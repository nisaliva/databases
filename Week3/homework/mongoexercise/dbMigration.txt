1- Insert records to mysql with using world.sql script.


2- Then exporting tables (city, country, countrylanguage) from world mysql database to csv files.

select * into outfile 'city.csv' FIELDS TERMINATED BY ',' LINES TERMINATED BY '\n' from city;

select * into outfile 'country.csv' FIELDS TERMINATED BY ',' LINES TERMINATED BY '\n' from country;

select * into outfile 'countrylanguage.csv' FIELDS TERMINATED BY ',' LINES TERMINATED BY '\n' from countrylanguage;

3- Then import data from csv to mongodb

mongoimport --db=new_world --collection=city --type=csv --headerline --file=/city.csv
mongoimport --db=new_world --collection=country --type=csv --headerline --file=/country.csv
mongoimport --db=new_world --collection=countrylanguage --type=csv --headerline --file=/countrylanguage.csv
